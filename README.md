# Prime Video Dashboard – Power BI Project
This project analyzes Amazon Prime Video’s global catalog using Power BI.
The dashboard highlights trends in content distribution, genres, ratings, geography, and release history from 1920–2021.
________________________________________
#Objective
To build an interactive dashboard that answers key questions about Prime Video’s content library, including:
•	What types of content dominate the platform?
•	Which genres and ratings appear most frequently?
•	How is content distributed across countries?
•	How has content production changed over time?
________________________________________
# Dataset Summary
•	Total Titles: 9,687
•	Genres: 519
•	Ratings: 25
•	Directors: 5,771
•	Timeline: 1920–2021
•	Types: Movies & TV Shows
•	Countries: 40+
________________________________________
# KPIs & Questions Answered
•	Total Titles
•	Movies vs TV Shows
•	Top Genres & Ratings
•	Titles by Country
•	Yearly Release Trends
________________________________________
# Process
Data Cleaning
•	Standardized genre and rating fields
•	Validated release years
•	Cleaned country names for map visuals
Data Modeling
•	Single-table model optimized for filtering
•	DAX measures for totals, percentages, and category rankings
Dashboard Features
•	KPI cards
•	Bar charts (Genres, Ratings)
•	Donut chart (Movies vs TV Shows)
•	World map (Country distribution)
•	Line chart (Release year growth)
________________________________________
# Key Insights
•	Movies dominate the catalog: ~81%
•	Drama is the most represented genre
•	Ratings 13+ and 16+ appear most frequently
•	Significant growth in content after 2000, especially post-2010
•	Strong geographic representation from US, UK, India, Canada, Germany, and Japan
________________________________________
# Conclusion
The Prime Video dashboard offers a clear overview of content diversity, distribution, and growth trends.
It demonstrates skills in Power BI, DAX, data cleaning, and dashboard storytelling, and serves as a complete BI analysis project.
